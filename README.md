# Bienvenido a simPOS

Este es un proyecto que utiliza **React** y TypeScript para construir aplicaciones web eficientes.

### Características principales:
- Construido con **React** para una experiencia de usuario interactiva.
- Utiliza **TypeScript** en un 73% del código base, lo cual mejora la escalabilidad y la mantenibilidad del proyecto.

### Comenzando

Para comenzar a utilizar este proyecto, clona el repositorio y ejecuta los siguientes comandos:

```bash
npm install
npm start
```
